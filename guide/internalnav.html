Internal Page Navigation

Internal page navigation for pages with long contents.

To use a sub navigation for long content pages, add the with-subnav class to the <div> element. Any <h2> element inside the <div> element will be used by the JavaScript to create the sub menu box. The JavaScript will automatically anchor each menu item to the contents.

Warning: because of structural changes and the way iFrame is used in the new version of D2L, the internal page navigation component has complications working inside D2L's environment. It is working as intended outside of D2L. If fixed for D2L, it will be broken outside of D2L. Since Droplets is not just for D2L's "special" environment, no changes will be made. Avoid using this component in D2L.

The <h2> elements are required for JavaScript. They will be used to create the sub menu navigation.
