Popover

A small overlay of content that appears when the mouse clicks on an icon, image, or text in page content. Do not use this component for tooltips and on hyperlinks.

Popover has four options: top, bottom, right, and left. Each option will be explained below with code and examples. To use popover, add the with-popover class and the data-title attribute to the HTML elements including <span>, <img>, etc. The data-title attribute is used to hold the popover content. Thus, it is required.

The <a> and <abbr> elements do not support the with-popover class.

Tip: use popover for non-essential information. Popover adds nice interaction to the content, but excessive use will cause unintended learning problems.

Accessibility: popover is not optimized for accessibility. Please avoid using popover if accessibility is a need.

Top

By default, the popover will be displayed on the top of the element. The top class is optional.

See the Pen Popover Top by Ethan Lin (@Lin87) on CodePen.

Bottom

The popover will be placed at the bottom of the text. To specify the bottom position, add the bottom class.

See the Pen Popover Bottom by Ethan Lin (@Lin87) on CodePen.

Right

The popover will be placed at the bottom of the text. To specify the right position, add the right class.

See the Pen Popover Right by Ethan Lin (@Lin87) on CodePen.

Left

The popover will be placed at the bottom of the text. To specify the left position, add the left class.

See the Pen Popover Left by Ethan Lin (@Lin87) on CodePen.

Image

Popover can be used on images too!

See the Pen Popover Image by Ethan Lin (@Lin87) on CodePen.

Popover for image will not be displayed when printed. This is a limitation of CSS3. Avoid using image with popover for essential information. Use image caption instead.
