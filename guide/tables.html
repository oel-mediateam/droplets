Tables

For tabular data only!

Basic Table

The default width of the <table> element will always be 100%; thus, it fills up the whole width of the page or container. The <table> is a block-level element. The use of <thead> is optional but recommended. The use of <tbody> is required in table with classes. The cells of the table will auto-resize accordingly to the content they held.

See the Pen Table by Ethan Lin (@Lin87) on CodePen.

Striped Table

Add "zebra-striping" to odd table rows within the <tbody> element by using the striped class.

See the Pen Striped Table by Ethan Lin (@Lin87) on CodePen.

Striped Multi-body Table

Add "zebra-striping" to even table body (<tbody>) within the <table> element by using the striped-body class. This class is recommended to use for table with rowspan.

See the Pen Striped Multi-body Table by Ethan Lin (@Lin87) on CodePen.

Table with Contextual Classes

Contextual classes are for presentational uses. They will not affect accessibility; thus, they are meaningless to accessibility software. In order words, they are for visual viewing only.

Cell Alignments

Horizontal Left DEFAULT

Left align the text in the cell by using the horz-left class.

See the Pen Table Cell Alignment HL by Ethan Lin (@Lin87) on CodePen.

Horizontal Center

Left align the text in the cell by using the horz-center class.

See the Pen Table Cell Alignment HC by Ethan Lin (@Lin87) on CodePen.

Horizontal Right

Left align the text in the cell by using the horz-right class.

See the Pen Table Cell Alignment HR by Ethan Lin (@Lin87) on CodePen.

Vertical Top DEFAULT

Left align the text in the cell by using the vert-top class.

See the Pen Table Cell Alignment VT by Ethan Lin (@Lin87) on CodePen.

Vertical Center

Left align the text in the cell by using the vert-center class.

See the Pen Table Cell Alignment VC by Ethan Lin (@Lin87) on CodePen.

Vertical Bottom

Left align the text in the cell by using the vert-bottom class.

See the Pen Table Cell Alignment VB by Ethan Lin (@Lin87) on CodePen.

Alignment classes can be applied to the table element as well. By applying the alignment class to the table element, every cell in the table will be aligned accordingly to the alignment.

Minimal Padding

To minimize the paddings inside each table cell, applied the minimal-padding class to the table element.

Smaller Text

To reduce the font size inside each table cell, applied the small-text class to the table element. The font size will be set to 14 pixels.

Borders Thickness and Style

Top Border

Apply the class top-border to the table cells or rows to display the top border.

Left Border

Apply the class left-border to the table cells or rows to display the left border.

Right Border

Apply the class right-border to the table cells or rows to display the right border.

Bottom Border

Apply the class bottom-border to the table cells or rows to display the bottom border.

Thickness

By default, the thickness of the border is one pixel. To the adjust the thickness of the border, two classes are available: medium or heavy.

Styles

By default, the style of the border is solid. To the adjust the style of the border, two classes are available: dotted or double-line.

Double line style only applicable to bottom border.

Cell Text Indentations

Single

To indent the text in a cell, apply the indent class.

Double

To double indent the text in a cell, apply the double-indent class.

See the Pen Borders with Variant Thickness by Ethan Lin (@Lin87) on CodePen.

Background Colors

Change the background of rows or individual cells with a meaningful color by using the following class: info (light blue), success (green), warning (yellow), danger (red), and neutral (light gray). Please use these contextual classes wisely.

See the Pen Table Cell Colors by Ethan Lin (@Lin87) on CodePen.
