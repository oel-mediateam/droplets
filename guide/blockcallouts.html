Block-Level Callouts

You would not want to miss it!

Horizontal

Horizontal callout uses a <div> element with the callout class. For callout heading, use <h4> element. For the content, use <p> element. For different variations of callout, append any of the following classes: success, info, warning, or danger to the <div> element's existing class attribute.

Icons can be used with callouts. Please refer to the icons page for a complete list of available icons.

Please refer to the best practice guide for the correct use of icons.

See the Pen Horizontal Callouts by Ethan Lin (@Lin87) on CodePen.

Vertical

Horizontal becomes vertical by applying the vertical class. Vertical callouts are best used for columned layouts or floatings.

See the Pen Vertical Callouts by Ethan Lin (@Lin87) on CodePen.

Minimal

Minimal callouts use callout class appended with a minimal class. Minimal class removed the fill color and the right border. To apply shadow, add the shadowed class.

See the Pen Minimal Callouts by Ethan Lin (@Lin87) on CodePen.

Alternative

An alternative minimal callout uses callout and minimal classes appended with a bordered class.

See the Pen Alternative Minimal Callouts by Ethan Lin (@Lin87) on CodePen.

Floating

Floating callout is for text wrap purposes. Only work with vertical callouts, and the width is fixed.

Left

Append the left class to the callout's <div> element to float the callout to the left.

See the Pen Callout Float Left by Ethan Lin (@Lin87) on CodePen.

Right

Append the right class to the callout's <div> element to float the callout to the right.

See the Pen Callout Float Right by Ethan Lin (@Lin87) on CodePen.

If you are experiencing alignment issues, place a <div> tag with the clearfix class after the place where you would like the elements to stop floating or aligning next to each other. For example: <div class="clearfix"></div>. The clearfix class only works if alignment classes are used.
