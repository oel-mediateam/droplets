Icons

Spice up headers, page contents, and callouts with intuitive icons.

The following icons are fonts and can be treated like regular text fonts. Please use the icons responsibly.

For Collaborative Programs

To use a icon, use a span element with the class name of the icon listed below. For example, <span class="icon-calendar"></span>.

Icon Sizes

Adjust the size of the icons with classes. The size of the icon stays relative to its parent element.

Size Class HTML Code Example
Default -- <span class="icon-calendar"></span>
Medium medium <span class="icon-calendar medium"></span>
Large large <span class="icon-calendar large"></span>
